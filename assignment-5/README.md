# Assignment 5 - Travel Agency API

"Vagabond Travels" - Travel agency backend API.

## Technologies
- Node.js
- Express.js
- MongoDB (Mongoose)
- REST API
- CORS

## Features
- REST API endpoints for destinations
- Customer inquiry submission
- Backend API for travel management

## Run
```bash
cd backend
npm install
npm start
```

## API Endpoints
- `GET /api/destinations` - List all destinations
- `POST /api/inquiries` - Submit inquiry
