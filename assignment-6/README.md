# Assignment 6 - Vehicle Marketplace

"AutoX" - Full-stack marketplace for buying/selling vehicles.

## Technologies
### Frontend
- React 18
- Vite
- React Router
- Axios
- Framer Motion
- Lucide React

### Backend
- Node.js
- Express.js
- MongoDB
- JWT (Authentication)
- Bcrypt.js
- CORS

## Features
- User authentication (JWT)
- Listing management for cars and bikes
- Appointment booking system
- Seller portal

## Run
```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
```
